# Git Hooks

- powerful built-in scripts
- automate actions by running automatically
- executes when particular event occurs.

**DIR:** .git/hooks/
- populated with series of sample scripts.

## Types of hooks

1. Client-side hooks
   1. Runs on local machine
   2. triggered by local actions
   3. enforce personal policies
   4. catch mistakes early
2. Server-side hooks
   1. Runs on remote repoistory
   2. enforce global policies
   3. for the entire team
   4. for an organization
