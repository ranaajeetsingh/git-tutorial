# git alias

1. custom shortcut for a single command
2. frequently used git command
3. one-time setup
4. faster and efficient workflow

**To create a alias**

`git config --global alias.<alias_name> <command>`

e.g. `git config --global alias.st status`

e.g. `git config --global alias.lg "log --grap --decorate --oneline --all`

**To list alias**

`git config --global --list`

**To remove alias**

`git config --global --unset alias.st`
