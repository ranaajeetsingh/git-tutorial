# 1. Removing untracked files

**Dry run command for files**

`git clean -n`

**Force delete command for files**

`git clean -f`

**Dry run command for both files and directories**

`git clean -nfd`

**Force delete command to delete both files and directories**

`git clean -fd`

# 2. Pruning old branches

**Remove any remote tracking branches which is no longer exists**

`git fetch --prune`

**To check merged branches**

`git branch --merged`

**To delete the branch**

`git branch -D <branch_name>`
