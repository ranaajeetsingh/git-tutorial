# Gitignore

1. Save lot of headaches
2. Keep repositories clean
3. Prevent accidental commits or unwanted files

## Why use gitignore?

1. Noise reduction
2. Sensitive information
3. Platform or OS specific files
4. Optimizing repository size

## How gitignore works?

1. Plain text file
2. Which files or directories to ignore
3. Placed in root directory of git repo
