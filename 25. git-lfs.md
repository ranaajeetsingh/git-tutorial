# Git LFS (Large File Storage)

Git max file size: 100 mb

## Git LFS

- changes how git stores the file
- small pointer file
- repository will be small and fast
- allows you to version large assests

1. install git lfs on local
2. set lfs on local: `git lfs install`
3. git lfs track <filename>
4. git lfs automatically created .gitattributes files to track lfs files.
5. .gitattributes file must need to be committed to work lfs properly.
